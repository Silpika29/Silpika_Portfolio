# ✨ Silpika Jattimyni | AI/ML Engineer Portfolio

A premium, interactive, and high-end personal portfolio website built to showcase my expertise, experience, and projects in Artificial Intelligence and Machine Learning.

## 🚀 Features
- **High-End Aesthetics:** Premium design system with elegant typography (Playfair Display & Source Sans 3), smooth micro-interactions, and a sophisticated color palette (Forest Green, Sage, Terracotta).
- **Interactive UI/UX:** 
  - Custom fluid mouse cursor with magnetic hover effects.
  - 3D parallax illustration cards with dynamic glass highlights.
  - Smooth page transitions and scroll-triggered animations.
- **Modern Tech Stack:** Built with React, Vite, TypeScript, and Tailwind CSS.
- **Responsive Design:** Flawlessly responsive across all devices, ensuring perfect viewing on mobile, tablet, and desktop.

## 🛠️ Built With
- **[React](https://reactjs.org/)** - UI Library
- **[Vite](https://vitejs.dev/)** - Next Generation Frontend Tooling
- **[Tailwind CSS](https://tailwindcss.com/)** - Utility-first CSS framework
- **[Framer Motion](https://www.framer.com/motion/)** - Animation library for React
- **[TypeScript](https://www.typescriptlang.org/)** - Static typing for JavaScript

## 📂 Project Structure
- `src/components/` - Reusable UI components (Hero, Navbar, CustomCursor, Footer, etc.)
- `src/pages/` - Core pages (Home, About, Skills, Experience, Projects, Awards)
- `src/assets/` - Static assets and image sequences
- `src/layouts/` - Main layout wrappers
